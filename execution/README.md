# Execution

Deterministic scripts for deployment, builds, and maintenance tasks.

## Scripts

| Script | Location | Purpose |
|--------|----------|---------|
| `deploy.sh` | Project root (`../deploy.sh`) | Full VPS deployment automation |

## VPS Helper Scripts

These are generated by `deploy.sh` on the VPS at `~/Accountant/`:

| Script | Purpose |
|--------|---------|
| `start.sh` | Start all services (backend, LiveKit, Hocuspocus) |
| `stop.sh` | Stop all services gracefully |
| `update.sh` | Pull latest code, rebuild frontend, restart services |
| `logs.sh` | View logs for all services |
| `status.sh` | Check service status and ports |

## Adding New Scripts

1. Place deterministic execution scripts in this directory
2. Reference them from the corresponding directive in `directives/`
3. Scripts must include input validation and error handling
4. Scripts must emit structured output for observability
